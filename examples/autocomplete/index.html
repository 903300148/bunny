Default Bootstrap 4 dropdown styles are used for autocomplete.

Free online country REST API https://restcountries.eu/ is used in this example to get data via AJAX.

AJAX url should contain {search} string in URL.

input must be inside a dropdown position: relative container with the same ID + "_autocomplete"

BunnyJS Autocomplete example

Start typing country to get autocomplete.

This input should be hidden and will be automatically filled with country ID when item picked from autocomplete.
This value can be used to store data in database and to work with relationships.
